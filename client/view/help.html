Experiment files operations

WellReader deals with experiments made using Tecan microplate reader (TMR).

In order to be processed using WellReader, the following constraints must be met:

Before processing an experiment using WellReader, it has to be converted to CSV format.

Allowed characters in an experiment file names are: a-z, A-Z, 0-9, _ and -.

Experiment storage

In WellReader, experiments files are not stored locally on your are disk. They must be uploaded on WellReader server before first processing. once the experiment is uploaded, it resides on the server and any modification of the experiment data is performed on the server.

At any time, you can download the actual version of an experiment file. It is even advised to do so, since Wellreader server may crash.

Experiment upload

To upload an experiment:

1. Right-click on the experiments node, and select Upload a Tecan CSV experiment file.

The following window appears:

2. Click on Select Tecan file to upload (.csv) and choose the CSV file corresponding to the experiment you want to upload, or drag a CSV file from your file explorer and drop it into the dashed rectangle.

Depending on the Internet connection speed and the size of the CSV file, the upload process may take more or less time. Once uploaded, the uploaded experiment appears in the Experiments list.

3. Click the experiment node to develop the experiment operations in the Experiments list.

Experiment download

Once an experiment is uploaded on the server, you can download it anytime.

To download an experiment:

1. Right-click on the experiment node, and select Download Well Reader (JSON) experiment file.

Experiments are internally stored using a Javascript format called JSON.

Experiment re-upload

Normally, this operation should be seldom used, since an experiment is stored on the server and stays there, unless you delete it. However, if you have inadvertently deleted an experiment, or want to revert to a previous version of an experiment, you may wish to re-upload a previously downloaded version of the experiment.

To re-upload an experiment:

1. Right-click on the experiment node, and select Upload a Well Reader (JSON) experiment file.

2. Click on Select JSON file to upload (.json) and choose the JSON file corresponding to the experiment you want to upload, or drag a JSON file from your file explorer and drop it into the dashed rectangle.

Delete an experiment

Before tou delete an experiment, you should download it so as to be able to restore it.

To delete an experiment:

1. Right-click on the experiment node to delete, and select Delete experiment.

Rename an experiment

To rename an experiment:

1. Right-click on the experiment node to rename, and select Rename experiment.

2. Modify the name of the experiment. Allowed characters in an experiment name are: a-z, A-Z, 0-9, _ and -.

3. Type Enter to validate the change.
